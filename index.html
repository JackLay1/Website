2022

Noise Generator

Written in Java, this is a simple project that will generate random noise. To accomplish this it first will set a base layer to a color, then generates 2 random numbers to form a coordinate that corresponds to a pixel on the screen. Every time a pixel is selected its alpha layer is increased. It will let you customize the number of attempts, as well as the size of the image. If you want you can save the image. Additionally it exports a list of coordinates. This served as a way to strenghten skills in I/O, GUIs, and data structures.

Analyzing Patterns in Sports Gambling

This was a data analysis project. Leveraging python, and using common data science libraries Pandas and SciPy for multilinear regression I wrangled and analyzed nba betting data from '07-'08 through '22-'23 NBA seasons to see if betting on the point spread is actually more profitable than betting on the moneyline underdog. This project was in an applied data science class where we went through the entire process of hypothesis formation and testing, actual analysis of results and confirmation/denial of your hypothesis, and a formal presentation of findings. You can view the slide deck here.
